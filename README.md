## 功能
* 定位
* 搜索城市
* 显示天气信息（包括实况天气，逐小时预报，每天预报）
* 显示生活指数（包括运动，洗车等指数）
* 存储查询过的城市，并可手动删除
## 使用的开源库
* [OkHttp](https://github.com/square/okhttp)
* [GSON](https://github.com/google/gson)
* [Litepal](https://github.com/LitePalFramework/LitePal)
* [Glide](https://github.com/bumptech/glide)
